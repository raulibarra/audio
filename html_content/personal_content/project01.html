Simple FPS & MetaSounds Demo

3D FPS Game (Unreal | C++ | Blueprints | MetaSounds)
  • Composed music and sound design for weapon, impact, footstep and die animations.
  • Designed an interactive horizontal resequencing music system using metasounds, with transitions synchronised to the tempo/beat of the music.
  • Custom metasound nodes for playing random sfxs, re-implementing some audio middleware features like random containers and random pitch/volume.
  • Integrated animations with event notifiers to fire FootStep SFXs.
  • Integrated dynamic mixing and ducking using custom sound classes.

I also worked on the gameplay logic, enemy AI and UI using blueprints and C++ classes.
